# [Remember-Helper](../../Tools/Remember-Helper/)

[背诵助手](../../Tools/Remember-Helper.exe)

- 语言: C++
- 版本: 0.1.0

## 使用说明

1. 在终端中运行 `Remember-Helper.exe` 命令

## [配置](../../Tools/Remember-Helper.yaml)

- root:
    - 名称: 根目录
    - 类型: string
    - 默认值: ./
- remember:
    - 名称: 背诵配置目录
    - 类型: string
    - 默认值: remember
- english:
    - 名称: 英语目录
    - 类型: string
    - 默认值: english
- chinese:
    - 名称: 语文目录
    - 类型: string
    - 默认值: chinese
- resources:
    - 名称: 资源目录
    - 类型: string
    - 默认值: Resources
